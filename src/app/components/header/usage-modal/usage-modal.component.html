つかいかた

  1. 「問題文」に問題の全文を入力
  2. 「問題の漢字」に問題にしたい部分の文字を入力
  3. 「よみがな」に問題にしたい部分のよみがなを入力
  4. 「読み/書き」で読み問題・書き問題を選択
  5. 「作成する」ボタンをクリック

つかいかたのコツ

問題文をきれいに表示するためには以下の長さに納めるのがオススメです。

この長さを超えると、読み問題の場合はフォントが小さくなります。

また、書き問題の場合は動作を停止する場合があります。その場合はリロードしてください。

きれいに表示できる文字数を超過した場合、警告が表示されます。

  • 10問以下の場合
    • 読み問題の場合：約29文字
    • 下記問題の場合：29-(問題部分の文字数×2)文字
  • 11問以上の場合
    • 読み問題の場合：約13文字
    • 書き問題の場合：13-(問題部分の文字数×2)文字

漢字以外も入力可能

実は「問題文」「問題の漢字」には漢字以外も入力できます。

ただし「問題の漢字」には「問題文」に含まれている文字が入力されている必要があります。

いろいろな文字種で読み・書き問題を作ってみよう！
